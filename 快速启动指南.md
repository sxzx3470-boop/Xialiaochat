# 快速启动指南

## 方案一：使用演示版（最简单，无需安装）

**直接双击打开 `demo.html` 文件即可使用！**

演示版功能：
- ✅ 用户注册/登录
- ✅ 搜索和添加好友  
- ✅ 实时聊天
- ✅ 好友列表管理
- ✅ 数据保存在浏览器本地

**使用方法：**
1. 双击 `demo.html` 在浏览器中打开
2. 注册两个账号（例如：user1 和 user2）
3. 用 user1 搜索并添加 user2 为好友
4. 用另一个浏览器窗口登录 user2，接受好友请求
5. 开始聊天！

## 方案二：使用完整版（需要 Node.js）

### 前提条件
- 已安装 Node.js（您说安装在 D 盘）

### 启动步骤

#### 方法 1：使用启动脚本
1. 双击 `start.bat` 文件
2. 等待依赖安装完成
3. 服务器会自动启动

#### 方法 2：手动启动
1. 打开命令提示符（cmd）或 PowerShell
2. 依次运行以下命令：

```bash
cd /d E:\openclaw
npm install
node server.js
```

3. 看到 "服务器运行在 http://localhost:3000" 表示成功

#### 如果 npm 命令不可用

如果提示找不到 npm，说明 Node.js 没有添加到环境变量，请：

1. 找到 Node.js 实际安装目录（通常在 `D:\nodejs` 或 `D:\Program Files\nodejs`）
2. 使用完整路径运行：

```bash
cd /d E:\openclaw
"D:\nodejs\npm" install
"D:\nodejs\node" server.js
```

### 访问应用

启动成功后，打开浏览器访问：http://localhost:3000

### 测试步骤

1. 打开浏览器窗口 A，注册账号：test1
2. 打开另一个浏览器窗口 B，注册账号：test2
3. 在窗口 A 中搜索 test2，发送好友请求
4. 在窗口 B 中会收到好友请求通知，点击"接受"
5. 在窗口 A 中点击 test2 好友，开始聊天
6. 在窗口 B 中可以看到收到的消息

## 常见问题

**Q: 演示版的数据会丢失吗？**
A: 演示版数据保存在浏览器 localStorage 中，清除浏览器缓存会丢失数据。

**Q: 完整版的服务器重启后数据会怎样？**
A: 当前版本使用内存存储，重启服务器数据会丢失。生产环境建议连接数据库。

**Q: 如何添加更多功能？**
A: 可以基于现有代码扩展，如群聊、发送图片、语音等功能。
