# 微信聊天软件

这是一个类似微信的聊天应用，支持加好友和实时聊天功能。

## 技术栈

- **后端**: Node.js + Express + Socket.IO
- **前端**: HTML5 + CSS3 + JavaScript
- **数据库**: 内存存储（可扩展为 MongoDB/MySQL）

## 功能特性

✅ 用户注册/登录
✅ 搜索和添加好友
✅ 实时一对一聊天
✅ 好友列表管理
✅ 在线状态显示
✅ 好友请求通知

## 安装说明

### 方法一：使用 Node.js（推荐）

1. **安装 Node.js**
   - 访问 https://nodejs.org/ 下载并安装 Node.js

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动服务器**
   ```bash
   npm start
   ```

4. **访问应用**
   - 打开浏览器访问：http://localhost:3000

### 方法二：单机演示版

我还可以创建一个使用 localStorage 的单机演示版本，无需安装 Node.js，直接在浏览器中运行。

## 使用说明

### 1. 注册账号
- 打开应用后，点击"注册"标签
- 输入用户名和密码
- 点击"注册"按钮

### 2. 登录
- 输入用户名和密码
- 点击"登录"按钮

### 3. 添加好友
- 在搜索框中输入要添加的用户名
- 点击"搜索"按钮
- 确认添加好友

### 4. 聊天
- 在好友列表中点击好友
- 在输入框中输入消息
- 按回车或点击"发送"按钮

### 5. 处理好友请求
- 当收到好友请求时，会弹出通知
- 点击"接受"或"拒绝"

## 测试示例

1. 打开两个浏览器窗口
2. 在第一个窗口注册账号：user1
3. 在第二个窗口注册账号：user2
4. 使用 user1 搜索并添加 user2 为好友
5. 使用 user2 接受好友请求
6. 开始聊天！

## 注意事项

- 当前版本使用内存存储，重启服务器后数据会丢失
- 生产环境建议使用 MongoDB 或 MySQL 作为数据库
- 建议添加更多的安全验证和错误处理
